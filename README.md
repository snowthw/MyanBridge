# MyanBridge

**MyanBridge** is a user-informed transliteration framework designed to improve Myanmar language input accessibility through **MyanBridge Transliteration Rules (MTR)**.

The project explores how Latin-script input can be transformed into Myanmar text while preserving usability, familiarity and accessibility for non-native learners and multilingual users.

## Research Motivation

Myanmar language remains underrepresented in multilingual computing and NLP due to challenges including:

- Difficult script input
- Inconsistent romanization practices
- Limited language datasets
- Lack of explicit word spacing

MyanBridge aims to reduce these barriers and support future low-resource Myanmar language technologies.

## Current Features

- User-informed transliteration rules (MTR)
- Latin-script → Myanmar text conversion
- Fixed mapping prototype
- Keyboard interface prototype
- Ambiguity reduction mechanisms

## Future Development

- AI-assisted transliteration prediction
- Suggestion and correction mechanisms
- Usability evaluation with learners
- Expansion of MTR rule coverage
- Applications for Myanmar NLP and educational tools
